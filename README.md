## Key Features

- **87% SEO task automation** via 8 AI agents
- **Execution layer**: Gmail API, Exa AI, Tavily, SerpAPI
- **Multi-engine**: Google, Bing, Yandex, Naver
- **AEO/GEO optimization**: Dual scoring (Google + AI Readiness)
- **Link building automation**: HARO, broken link, guest post, unlinked mention
- **Campaign tracker**: Draft → Sent → Replied → Negotiating → Live
- **Enterprise security**: OAuth 2.0, MFA, RBAC, encryption, WAF
- **50+ frontend pages**: claude-design principles, anti-slop
